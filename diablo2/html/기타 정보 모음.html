☞ 용병 정리 ☞ 라주크 소켓퀘시 소켓수
1. combat(전투) : 노멀/지옥 -  기도(프레이어) // 악몽(나메) - 가시(쏜즈)  - 흰색 : 해당 아이템의 최대 소켓수 (난이도별 별도)
2. defence(수비): 노멀/지옥 -  인내(디파이언스) // 악몽(나메) -  신성한빙결(홀리프리즈)  - 매직 : 1~2솟 랜덤
3. offense(공격): 노멀/지옥 - 축복받은조준(블레시드에임) // 악몽(나메) - 위세(마이트)  - 레어, 유닉 : 1솟
　
- 기도(프레이어) : 자신과 파티원들의 생명력을 회복 (맨땅 초반용) 4솟무기(영혼) 재료는 노멀 젖소방, 액트5 용으로 사용
- 가시(쏜즈) : 피격시 일정비율의 데미지를 반사 (사용하지 않음) * 크리스탈소드, 브로드소드 (단 보스몹 드랍템은 5~6솟이 뚤림)
- 인내(디파이언스) : 자신과 파티원들의 방어력 상승 (딮위주셋팅) 　
- 신성한빙결(홀리 프리즈) : 자신 주변의 몬스터들을 빙결 콜드 피해를 주고 움직임과 공격속도를 둔화(마법캐릭) ☞ 스킬, 스텟 초기화 방법
- 축복받은조준(블레스드 에임) : 자신과 파티원들의 공격등급(어레)를 향상시킨다 (전투캐릭) 1. 퀘스트로 각 난이도별 1번씩 총 3번가능 
- 위세(마이트) : 자신과 파티원들의 물리 공격력을 향상시킨다 (전투캐릭 사용) 2. 면죄의 증표로 무한가능
 - 안다리엘, 메피스토, 디아블로, 바알이 드랍한 정수를 조합 = 면죄의증표

3 주울룬　+ 1 최하급 토파즈　　　=  앰룬 　
3 앰룬　　+ 1 최하급 자수정　　　=  솔룬 ☞ 유니크아이템 업그레이드 정보
3 솔룬　　+ 1 최하급 사파이어　　=  샤에룬  - 무기 
3 샤에룬　+ 1 최하급 루비　　　　=  돌룬 노멀 -> 익셉 : 랄 + 솔 + 최상급에메랄드 
3 돌룬　　+ 1 최하급 에메랄드　　=  헬룬 익셉 -> 엘리트 : 룸 + 풀 + 최상급에메랄드 
3 헬룬　　+ 1 최하급 다이아몬드　=  이오룬 - 방어구
3 이오룬　+ 1 하급 토파즈　　　　=  룸룬 노멀 -> 익셉 : 탈 + 샤에 + 최상급다이아몬드 
3 룸룬　　+ 1 하급 자수정　　　　=  코룬 익셉 -> 엘리트 : 코 + 렘 + 최상급다이아몬드
3 코룬　　+ 1 하급 사파이어　　　=  팔룬 　
3 팔룬　　+ 1 하급 루비　　　　　=  렘룬 ☞  레어아이템 업그레이드
3 렘룬　　+ 1 하급 에메랄드　　　=  풀룬 - 무기
2 풀룬　　+ 1 하급 다이아몬드　　=  움룬 노멀 -> 익셉 : 오르트 + 앰 + 최상급사파이어
2 움룬　　+ 1 토파즈 　　　　　　=  말룬 익셉 -> 엘리트 : 팔 + 우움 + 최상급사파이어
2 말룬　　+ 1 자수정 　　　　　　=  아이스트룬 - 방어구
2 이스트룬+ 1 사파이어 　　　　　=  굴룬 노멀 -> 익셉 : 랄 + 주울 + 최상급자수정
2 굴룬　　+ 1 루비 　　　　　　　=  벡스룬 익셉 -> 엘리트 : 코 + 풀 + 최상급자수정
2 벡스룬　+ 1 에메랄드　　　　　 =  오움룬 　
2 오움룬　+ 1 다이아몬드　　　　 =  로룬 ☞ 주요 크래프트 정보
2 로룬　　+ 1 상급 토파즈　　　　=  수르룬 * 크래프트는 주로 사용하는것 외엔 가치가 떨어짐
2 수르룬　+ 1 상급 자수정　　　　=  베르룬 * 캐릭렙과 재료(예:목걸이)의 숨은렙이 높아야 함
2 베르룬　+ 1 상급 사파이어　　　=  자룬 * 도박으로 재료를 뽑으면 +4렙이 높은게 뽑아짐
2 자룬　　+ 1 상급 루비　　　　　=  참룬 * 크래프트는 상급버전으로 업그레이드가 안됨
2 참룬　　+ 1 상급 에메랄드　　　=  조드룬
